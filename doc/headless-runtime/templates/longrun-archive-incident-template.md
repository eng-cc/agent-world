# Headless-Runtime 长稳归档与故障追溯模板

审计轮次: 4

## 目的
- 为 `TASK-NONVIEWER-003` 提供统一的长稳归档与故障追溯记录格式。
- 把运行区间、失败签名、归档路径、恢复动作与复盘结论收敛到同一模板。

## 使用说明
- 每次长稳 / soak / 线上故障复盘至少填写一份记录。
- 若缺少归档路径或失败签名，结论只能为 `blocked`。
- 归档可来自日志、summary、状态快照或故障现场保底落盘。
