# oasis7 Simulator：LLM 多场景评测基线（设计文档）设计

- 对应需求文档: `doc/world-simulator/llm/llm-multi-scenario-evaluation.prd.md`
- 对应项目管理文档: `doc/world-simulator/llm/llm-multi-scenario-evaluation.project.md`

## 1. 设计定位
定义 LLM 多场景评测基线设计，为不同世界场景建立统一输入、指标与复核方法。

## 2. 设计结构
- 场景基线层：选择代表性场景并固定输入集。
- 评测指标层：定义行为质量、稳定性和成本指标。
- 执行采样层：批量运行场景并记录模型输出。
- 复核对比层：支持版本间对比与失败定位。

## 3. 关键接口 / 入口
- 评测场景集
- 指标定义
- 批量执行入口
- 结果对比报告

## 4. 约束与边界
- 评测样本需可复现。
- 指标定义要跨场景可比。
- 不在本专题引入自动裁决替代人工复核。

## 5. 设计演进计划
- 先冻结场景样本。
- 再定义指标与执行流程。
- 最后形成版本对比基线。
