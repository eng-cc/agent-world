# Agent World Simulator：LLM 对话轮次驱动与右侧 Chat 面板（设计文档）设计

- 对应需求文档: `doc/world-simulator/llm/llm-dialogue-chat-loop.prd.md`
- 对应项目管理文档: `doc/world-simulator/llm/llm-dialogue-chat-loop.project.md`

## 1. 设计定位
定义 LLM 对话轮次驱动与右侧 Chat 面板设计，统一多轮对话节拍、消息展示与用户交互入口。

## 2. 设计结构
- 轮次驱动层：以稳定回合推进对话输入、生成和反馈。
- 面板布局层：右侧 Chat 面板承载历史、状态和输入区。
- 交互桥接层：把对话事件与运行时/工具调用联动。
- 状态恢复层：支持刷新、重连或重开会话后的历史恢复。

## 3. 关键接口 / 入口
- 对话轮次状态机
- 右侧 Chat 面板
- 对话事件桥接
- 历史恢复入口

## 4. 约束与边界
- 轮次推进需与运行时状态一致。
- 面板展示不得丢失关键生成与工具事件。
- 不在本专题扩展多会话工作台。

## 5. 设计演进计划
- 先固定对话回合模型。
- 再接右侧面板布局。
- 最后补状态恢复与回归。
