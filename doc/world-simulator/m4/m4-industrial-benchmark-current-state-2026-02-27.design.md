# M4 产业链借鉴点（结合当前实现现状，2026-02-27）设计

- 对应需求文档: `doc/world-simulator/m4/m4-industrial-benchmark-current-state-2026-02-27.prd.md`
- 对应项目管理文档: `doc/world-simulator/m4/m4-industrial-benchmark-current-state-2026-02-27.project.md`

## 1. 设计定位
定义 M4 产业链当前实现现状与借鉴点的基线设计，为后续工业经济专题提供统一比较框架。

## 2. 设计结构
- 现状基线层：记录当前工业链路、资源流与生产约束。
- 借鉴映射层：把外部参考点映射到当前实现缺口。
- 优先级排序层：识别最值得先收敛的工业问题。
- 演进承接层：将基线对接后续 m4 专题。

## 3. 关键接口 / 入口
- 当前实现基线
- 借鉴点映射
- 优先级清单
- 后续专题承接点

## 4. 约束与边界
- 基线记录需忠于当前实现。
- 借鉴点用于对照，不直接替代本项目设计。
- 不在本专题直接实现全部改造。

## 5. 设计演进计划
- 先冻结当前状态。
- 再建立借鉴映射。
- 最后输出优先级承接关系。
