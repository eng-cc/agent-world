# Viewer 视觉升级设计

- 对应需求文档: `doc/world-simulator/viewer/viewer-visual-upgrade.prd.md`
- 对应项目管理文档: `doc/world-simulator/viewer/viewer-visual-upgrade.project.md`

## 1. 设计定位
定义 Viewer 视觉表现从基础可见向更高质感和层次升级的整体方案：围绕主题、材质、构图和可读性同步收敛。

## 2. 设计结构
- 主题风格层：统一主题预设、语义色板和材质风格基线。
- 构图表现层：围绕主对象、背景和辅助参考物优化默认镜头与展示层级。
- 资产质量层：逐步提升 mesh、贴图和材质细节，避免同质化和低质感。
- 验证收口层：通过截图闭环和视觉基线持续验证升级效果。

## 3. 关键接口 / 入口
- 主题 preset
- 材质/贴图/mesh 资产入口
- 默认镜头与展示层级
- 截图闭环基线

## 4. 约束与边界
- 视觉升级不改变玩法规则和协议。
- 升级必须服务于可读性，不能只追求装饰性效果。
- 主题和资产升级要可渐进回滚，避免一次性大爆炸替换。
- 验证以可复跑视觉基线为准，避免纯主观口径。

## 5. 设计演进计划
- 先冻结视觉升级目标和基线。
- 再分主题、构图、资产三层推进。
- 最后用截图/样张基线收口升级效果。
