# Viewer 工业风可视化闭环设计

- 对应需求文档: `doc/world-simulator/viewer/viewer-industrial-visual-closure.prd.md`
- 对应项目管理文档: `doc/world-simulator/viewer/viewer-industrial-visual-closure.project.md`

## 1. 设计定位
定义工业风可视化缺口收敛方案：围绕产线、物流、经营、告警和产品化表现层，给 Viewer 建立“可定位、可量化、可回放”的工业经营观察界面。

## 2. 设计结构
- 产线物流层：基于现有 `WorldSnapshot + WorldEvent` 汇总产线实体、物流路由和吞吐指标。
- 经营驾驶舱层：围绕供需、成本、收益和库存健康构建经营摘要面板。
- 多尺度导航层：提供世界/区域/节点三级聚焦与告警根因链跳转。
- 产品化表现层：统一语义色板、关键态强调和可开关轻量动效。

## 3. 关键接口 / 入口
- `egui_right_panel.rs`
- `egui_observe_section_card.rs`
- `world_overlay.rs`
- 工业链路/告警摘要相关 UI 文本入口

## 4. 约束与边界
- 本轮以可视化表达增强为主，不新增玩家手操协议或重写内核经济规则。
- 新增面板必须模块化显隐，避免压缩既有调试信息。
- 代理指标与降级提示需要明确标识，避免把不完整数据伪装成精确经营指标。
- 表现层增强不能牺牲回归可测性，必须有截图或单测产物。

## 5. 设计演进计划
- 先冻结四个任务组的指标口径和入口。
- 再按 A~D 逐组补观察面板、导航和表现层。
- 最后通过截图闭环与回归测试完成工业可视化收口。
