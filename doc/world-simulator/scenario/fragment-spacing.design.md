# Agent World Simulator：小行星碎片最小间距（设计文档）设计

- 对应需求文档: `doc/world-simulator/scenario/fragment-spacing.prd.md`
- 对应项目管理文档: `doc/world-simulator/scenario/fragment-spacing.project.md`

## 1. 设计定位
定义小行星碎片最小间距设计，避免碎片重叠、过密生成与初始化碰撞。

## 2. 设计结构
- 间距规则层：定义碎片最小距离与密度限制。
- 生成校验层：在生成时对候选位置执行间距检测。
- 场景覆盖层：允许场景配置覆盖默认间距参数。
- 回归验证层：验证高密场景下仍保持可接受布局。

## 3. 关键接口 / 入口
- 最小间距配置
- 候选位置校验
- 场景覆盖字段
- 间距回归用例

## 4. 约束与边界
- 间距规则优先保证可生成和无重叠。
- 场景覆盖需显式可见。
- 不在本专题引入复杂物理碰撞求解。

## 5. 设计演进计划
- 先冻结最小间距规则。
- 再接入场景覆盖。
- 最后固化高密布局回归。
