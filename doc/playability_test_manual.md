# 一套**完整可落地的量化评分模型 + 雷达图指标结构**，配合playability_test_card使用。

目标：

* 把问卷 → 数值
* 把数值 → 6 个核心能力维度
* 把维度 → 雷达图
* 一眼看出结构性问题

---

# 🎮 一、总体结构

我们把问卷压缩为 6 个核心维度：

```
1️⃣ 理解清晰度（Clarity）
2️⃣ 控制感（Agency）
3️⃣ 策略深度（Strategy）
4️⃣ 可预测性（Predictability）
5️⃣ 语言机制强度（Language Impact）
6️⃣ 重玩意愿（Retention）
```

每个维度 0–100 分。

最终输出：

* 总分（0–100）
* 六维雷达图
* 风险标签

# 📈 三、六维评分公式

---

# 1️⃣ 理解清晰度（Clarity）

来源题目：

* Q1（目标理解）
* Q2（指令影响理解）
* Q3（角色理解）
* Q11（能解释发生了什么）

公式：

```
Clarity = (Q1 + Q2 + Q3 + Q11) / 20 × 100
```

满分 100。

⚠ 若 < 60 → 游戏说明结构问题。

---

# 2️⃣ 控制感（Agency）

来源题目：

* Q4（系统听懂）
* Q5（符合预期）
* Q6（成功归因）
* Q7（失败归因）

Q7 特殊处理：

| 选项     | 分值 |
| ------ | -- |
| 我策略有问题 | 5  |
| 运气不好   | 3  |
| 系统随机   | 1  |
| 系统没听懂  | 1  |
| 不确定    | 2  |

公式：

```
Agency = (Q4 + Q5 + Q6 + Q7) / 20 × 100
```

⚠ 若 < 65 → 玩家感觉“不是自己在玩”。

---

# 3️⃣ 策略深度（Strategy）

来源题目：

* Q8（是否形成策略）
* Q9（是否调整）
* Q17（是否愿意研究更优策略）

公式：

```
Strategy = (Q8 + Q9 + Q17) / 15 × 100
```

⚠ 若 < 60 → 伪策略风险。

---

# 4️⃣ 可预测性（Predictability）

来源题目：

* Q5（结果符合预期）
* Q12（能预测下一步）
* Q7（失败归因）

公式：

```
Predictability = (Q5 + Q12 + Q7) / 15 × 100
```

⚠ 若 < 55 → 玩家会说“随机”。

---

# 5️⃣ 语言机制强度（Language Impact）

来源题目：

* Q10（表达差异）
* Q23（是否优化表达）
* Q2（理解指令影响）

Q23 转换：

| 选项   | 分值 |
| ---- | -- |
| 没有   | 1  |
| 有一点  | 3  |
| 明显调整 | 5  |

公式：

```
LanguageImpact = (Q10 + Q23 + Q2) / 15 × 100
```

⚠ 若 < 60 → 语言只是输入框，不是玩法。

---

# 6️⃣ 重玩意愿（Retention）

来源题目：

* Q15（有趣程度）
* Q16（再玩一局）
* Q18（继续玩意愿）

公式：

```
Retention = (Q15 + Q16 + Q18) / 15 × 100
```

⚠ 若 < 70 → 上线风险高。

---

# 🧮 四、总分公式

建议加权：

```
TotalScore =
Clarity × 0.15 +
Agency × 0.25 +
Strategy × 0.2 +
Predictability × 0.15 +
LanguageImpact × 0.1 +
Retention × 0.15
```

原因：

控制感是核心，占比最高。

# 🚨 六、结构性风险判定规则

### 高风险情况 1

```
Agency < 60 且 Predictability < 55
```

→ 玩家认为游戏随机。

---

### 高风险情况 2

```
Strategy > 70 但 Retention < 60
```

→ 游戏“有深度但不好玩”。

---

### 高风险情况 3

```
LanguageImpact < 55
```

→ LLM 没成为机制核心。

---

# 🧠 七、你这个品类的健康结构

理想形态：

```
Clarity        ≥ 70
Agency         ≥ 75
Strategy       ≥ 70
Predictability ≥ 65
LanguageImpact ≥ 65
Retention      ≥ 75
```

如果：

* Agency 高
* Strategy 高
* LanguageImpact 高

→ 你成功做出了“对话型策略游戏”，而不是“AI模拟器”。

# 📊 八、批量样本分析建议

当你有 30+ 份问卷：

重点看：

* 平均值
* 标准差（是否分化）
* 不同玩家背景对比
* 熟悉LLM vs 不熟悉LLM

如果熟悉LLM玩家分高、普通玩家分低 → 设计门槛问题。
