```text
大家好，这里是 oasis7。

先发第一条，打个招呼。

最近我在做一个游戏。
更准确一点说，是我在和一群 agent 一起做。

这件事到现在都还会让我偶尔觉得有点怪。
因为很多时候确实不是“我想完了再让工具执行”，
而是真的在一起讨论，一起拆，一起往前做。

当然也不是一直都很顺。
有时候很好用，
有时候也会跑偏，
有时候能把人气得想重来。

但不管怎么样，这个项目做下来，感觉和我以前做东西很不一样。

所以这个号之后会慢慢记一点这件事。
一边记这个游戏，
一边记和 agent 一起做游戏到底是什么感觉。

先这样。
后面慢慢发。
```
