Playable Civilization Game

Game first on top screens, engine details in lower screens

`agent-world` is primarily a long-horizon civilization game where players make recurring strategic decisions. The lower sections then explain how a WASM engine and runtime guardrails make those loops extensible and auditable.

Core Gameplay

Top-half narrative is optimized for player understanding, not engine jargon.

Three-layer Loops

Immediate feedback on login, milestone outcomes in days, and civilization goals across weeks.

Value: sustained motivation and pacing

War and Governance Tension

Blockades, negotiations, voting, and institutional conflicts coexist in one world.

Value: strategic depth with real counterplay

Indirect Player Control

Players set goals and strategy boundaries; agents stay autonomous and outcomes stay consequential.

Value: high agency without deterministic scripting

Irreversible Civilizational History

Major events become persistent historical state that reshapes future play.

Value: progression feels meaningful and accumulative

How You Play

Daily 5-15 Minutes

Inspect world change, adjust goals, and react to opportunities or threats.

Phase 1-3 Days

Push infrastructure, alliances, or localized conflicts toward concrete milestones.

Long Run Weeks

Compete for governance power and rewrite the civilization-level balance.

Demo and Quick Start

Offline Gameplay Replay (Recommended)

Generate one world run and replay it to understand loop pacing and state transitions.

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_demo -- twin_region_bootstrap --out .data/world_viewer_data
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_server -- .data/world_viewer_data 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

Live Match Progression (LLM default, `--no-llm` fallback)

Run continuous world progression for long-run strategy observation and balancing.

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- llm_bootstrap --bind 127.0.0.1:5023 --web-bind 127.0.0.1:5011 --tick-ms 300
# Optional: fallback to script decisions
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- llm_bootstrap --no-llm --bind 127.0.0.1:5023 --web-bind 127.0.0.1:5011 --tick-ms 300
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5023

Three-step First Run

  1. Clone the repository and enter the workspace.
  2. Run `env -u RUSTC_WRAPPER cargo check` to validate environment setup.
  3. Replay one run first, then switch to live progression for longer horizons.

Need full parameters and troubleshooting? Open the documentation hub (EN).
