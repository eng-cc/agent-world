Playable Civilization War

This is not a simulator. It is a civilization war that remembers your choices.

Spend 5-15 minutes a day making strategic calls: reroute resources, swap alliances, or force policy change. The world keeps evolving, history does not rewind, and every decision reshapes your next conflict window.

Four System Pillars That Keep Players Invested

Player value first, implementation detail second. Each pillar maps to concrete consequence and long-horizon payoff.

Three Loops, One Strategic Arc

Short loops deliver instant response, mid loops resolve phase outcomes, and long loops write civilization history. You are not finishing isolated rounds; you are compounding leverage across time.

Value: retention pressure and emotional pacing stay active

War x Governance Interlock

Blockades, policy votes, and regional treaties run in parallel. Tactical play and institutional design collide continuously.

Value: strategic depth beyond combat

Indirect Player Control

You define intent, constraints, and priorities. Agents execute under uncertainty and outcomes remain irreversible.

Value: command-level agency without click fatigue

High Reward, Real Blowback

Aggressive expansion can trigger sanctions, trust collapse, and coalition counterplay. The system forces trade-offs instead of free snowballing.

Value: every win has a meaningful price

A Day in the War Room

You make fundamentally different decisions across three time scales inside one persistent world.

Daily Loop (5-15 Minutes)

Scan volatility, tune thresholds, and handle incidents before rivals seize tempo.

Phase Loop (1-3 Days)

Fight for strategic nodes, negotiate temporary pacts, and rewire production flow for phase advantage.

Season Loop (Weeks)

Push governance changes and lock tactical momentum into civilization-level rules.

Start Fast: Play First, Analyze Later

Offline Match Replay (Recommended)

Generate one world state and replay it to understand pacing and event causality.

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_demo -- twin_region_bootstrap --out .data/world_viewer_data
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_server -- .data/world_viewer_data 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

Live Continuous Match (LLM default)

Run the world continuously and observe multi-day strategic pull-and-counterpull.

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- llm_bootstrap --bind 127.0.0.1:5023 --web-bind 127.0.0.1:5011 --tick-ms 300
# Optional: fallback to scripted decisions
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- llm_bootstrap --no-llm --bind 127.0.0.1:5023 --web-bind 127.0.0.1:5011 --tick-ms 300
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5023

Three Steps to Your First Civilization Run

  1. Clone the repository and enter the workspace.
  2. Run `env -u RUSTC_WRAPPER cargo check` to validate setup.
  3. Replay once to learn the loop, then switch to live progression.

Full parameters, troubleshooting, and test matrix live in the documentation hub (EN).
