Persistent · Believable · Emergence-first

面向硅基文明的持久世界模拟器

`agent-world` 是一个“规则先行”的多主体仿真系统。Agent 在资源、制度与关系约束下持续演化， 世界状态可落盘、可恢复、可回放、可审计。

  • Persistent状态可持久化与恢复
  • Multi-agent独立个体持续协作/竞争
  • WASM Sandbox开放模块扩展能力

核心能力

持久世界

世界快照与事件日志可落盘，支持长时演化与实验复现实验结果。

多主体行为

每个 Agent 拥有独立状态、记忆与目标，在约束下形成涌现行为。

规则优先

Agent 行动必须通过世界规则校验，行为后果可追踪到资源与关系层。

LLM 行为接入

内置 OpenAI 兼容 `chat/completions` 决策接口，支持多步与 repair 机制。

可视化观测

提供 viewer 与 live server，支持离线回放、在线观测与诊断信息展示。

开放扩展

通过 Rust + WASM 模块接入新能力，持续拓展社会与经济系统模拟深度。

演示路径

离线回放

先生成 `snapshot.json` 与 `journal.json`，再启动 viewer server + UI 进行回放。

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_demo -- twin_region_bootstrap --out .data/world_viewer_data
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_server -- .data/world_viewer_data 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

在线模式（可选 LLM）

直接运行 live server 推动世界演化，viewer 实时订阅并渲染。

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- twin_region_bootstrap --bind 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

快速开始

  1. 克隆仓库并进入项目根目录。
  2. 执行 `env -u RUSTC_WRAPPER cargo test` 验证环境。
  3. 运行 demo 工具或 viewer 体验世界初始化与演化。

需要 LLM 驱动时，先从 `config.example.toml` 复制到 `config.toml`，并配置 `AGENT_WORLD_LLM_MODEL/BASE_URL/API_KEY`。
