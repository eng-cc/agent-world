核心能力

只保留最关键的四个能力块，让你在 1 分钟内判断项目是否匹配你的研究与工程目标。

规则执行内核

行为必须经过规则校验，拒绝越权动作并留下审计证据。

价值：保证可治理性与可解释性

持久化回放

快照 + 事件日志组合，支持实验恢复、对比与复盘。

价值：保证可复现与可验证

多主体世界建模

Agent 在资源、关系与制度约束下持续演化，不是一次性对话。

价值：观察协作、冲突与秩序形成

LLM / WASM 扩展接口

兼容 OpenAI 风格模型与模块化规则扩展，逐步提升世界复杂度。

价值：稳定底座上持续增强智能策略

架构闭环

Runtime

统一动作入口、策略校验与执行边界。

Simulator

推进世界时间与状态，沉淀快照和事件资产。

Viewer

离线回放与在线订阅双路径观测世界变化。

LLM/Modules

按需接入外部决策能力与可插拔规则模块。

应用场景

协作策略实验

对比制度差异下的资源效率、稳定性和冲突速率。

LLM 行为安全沙盒

在规则护栏内评估策略，快速定位越权路径和修复点。

教学与演示

用“规则 -> 行为 -> 后果”链路解释复杂系统行为。

演示与上手

离线回放（推荐）

先生成数据，再启动回放服务与 viewer，快速看到完整演化轨迹。

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_demo -- twin_region_bootstrap --out .data/world_viewer_data
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_server -- .data/world_viewer_data 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

在线演化（可选 LLM）

live server 持续推进世界，viewer 实时订阅状态与事件。

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- twin_region_bootstrap --bind 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

三步完成首跑

  1. 克隆仓库并进入目录。
  2. 执行 `env -u RUSTC_WRAPPER cargo test` 检查环境。
  3. 运行 demo 或 live 链路，观察事件与状态变化。
