可玩的文明战争

这不是模拟器，而是一场会记住你决定的文明战争

你每天只需 5~15 分钟做策略决策：调资源、换联盟、赌政策。世界会持续演化，历史不可回滚， 你的每一步都会影响下一阶段的冲突与秩序。首次本地运行需先完成 Rust 构建（常见 5~15 分钟，视机器而定）。

  • 关键抉择结盟还是制裁

  • 引擎能力刻意后置到下半屏，避免技术喧宾夺主

让人上头的四条系统支柱

先讲玩家价值，再讲实现细节。每条支柱都对应明确行为后果和长期投入回报。

三层循环统一驱动

短循环给即时反馈，中循环给阶段胜负，长循环沉淀文明史。你不会“打完一把就结束”，而是持续经营下一轮优势。

价值：留存动机与情绪曲线持续存在

战争 x 治理双线博弈

资源封锁、政体投票、区域条约同步发生，战术操作和制度设计相互缠绕。

价值：战斗之外有更高层策略空间

玩家间接控制

你定义目标、边界与底线，Agent 执行并承担不可逆后果，减少机械点击，放大决策张力。

价值：更像“指挥文明”，不是“点点点”

高收益伴随高反噬

越激进的扩张越容易触发围堵、制裁与信誉崩塌，系统会逼你做取舍，而不是一路滚雪球。

价值：每次胜利都必须付出真实代价

玩家的一天：从侦察到改写秩序

同一世界里，你会在三个时间尺度上做完全不同的决策。

日常循环（5~15 分钟）

查看资源波动、调整策略阈值、处理突发事件，确保今天不被对手偷节奏。

阶段循环（1~3 天）

争夺关键节点、签订短期协定、重构生产链，形成阶段性碾压或反扑。

赛季循环（数周）

推动制度变更与治理提案，把一次战术优势固化成文明级长期规则。

本地 3 步上手：先回放，再在线

离线战局回放（推荐）

快速生成世界数据并回放，先理解玩法节奏与关键事件链。

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_demo -- twin_region_bootstrap --out .data/world_viewer_data
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_server -- .data/world_viewer_data 127.0.0.1:5010
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5010

在线持续战局（LLM 默认）

进入持续推进模式，观察多日策略拉扯与对抗稳定性。

env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- llm_bootstrap --bind 127.0.0.1:5023 --web-bind 127.0.0.1:5011 --tick-ms 300
# 可选：回退脚本决策
env -u RUSTC_WRAPPER cargo run -p agent_world --bin world_viewer_live -- llm_bootstrap --no-llm --bind 127.0.0.1:5023 --web-bind 127.0.0.1:5011 --tick-ms 300
env -u RUSTC_WRAPPER cargo run -p agent_world_viewer -- 127.0.0.1:5023

三步完成本地首局（首次编译约 5~15 分钟）

  1. 克隆仓库并进入目录。
  2. 执行 `env -u RUSTC_WRAPPER cargo check` 验证环境。
  3. 先跑回放熟悉规则，再切到在线模式观察长期演化。

参数说明、排障路径与测试矩阵请前往 文档中心（中文）。

引擎底座（给工程读者）

如果你关注可扩展性与治理边界，这里是底层：玩法模块如何在 Runtime 约束下安全演进。

Gameplay Layer

战争、治理、经济等规则以 WASM 模块化加载与替换。

Runtime Kernel

统一约束资源守恒、时间单向、动作合法性，保证世界一致性。

Viewer & Tooling

离线回放、在线订阅、调试面板共享同一事件证据链。

Consensus & Governance

以提案机制定义模块升级与规则替换的边界与权限。
